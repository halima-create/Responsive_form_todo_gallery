# Responsive_form_todo_gallery

This project is a responsive and interactive web interface developed using *HTML, CSS, and JavaScript*. It demonstrates basic front-end development concepts including form validation, DOM manipulation, and responsive layouts.

## Features
- 📬 Contact Form with JavaScript validation for user input
- ✅ Interactive To-Do List to add and remove tasks dynamically
- 🖼️ Image Gallery that allows users to add images using URLs
- 📱 Responsive design using *CSS Flexbox and Grid*
- ⚡ Dynamic webpage interactions using *JavaScript DOM manipulation*

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript

## Project Structure
- index.html – Main webpage structure  
- styles.css – Styling and responsive layout  
- script.js – Interactive functionality and validation

## Purpose
This project was created as part of *Task 2 of the Web Development Internship at ApexPlanet* to practice building interactive and responsive web components using core front-end technologies.
